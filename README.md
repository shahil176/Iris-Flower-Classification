## Project Name

Iris Flower Classification

## Project Scope

The objective of this project is to build a Machine Learning model that can classify iris flowers into three species based on their sepal and petal measurements.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Google Colab

## Dataset Information

The Iris dataset contains 150 flower samples with the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Target Classes:

* Setosa
* Versicolor
* Virginica

## Methodology

1. Data Loading
2. Data Exploration
3. Data Visualization
4. Train-Test Split
5. Model Training using K-Nearest Neighbors (KNN)
6. Prediction
7. Accuracy Evaluation

## Results

* Dataset Size: 150 records
* Features: 4
* Accuracy Achieved: 100%

## Conclusion

The KNN model successfully classified iris flowers with excellent accuracy. This project demonstrates the basic workflow of a Machine Learning classification problem.
