## PROJECT SCOPE

The objective of this project is to develop a Machine Learning model capable of classifying iris flowers into their respective species based on sepal and petal measurements. This project demonstrates the implementation of a classification algorithm and the complete machine learning workflow, including data analysis, visualization, model training, and evaluation.

## TOOLS USED

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Google Colab

## DATASET

Iris Flower Dataset

## FEATURES USED

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

## TARGET CLASSES

* Iris Setosa
* Iris Versicolor
* Iris Virginica

## MODEL USED

K-Nearest Neighbors (KNN)

## ACCURACY

100%

## PROJECT OUTCOME

Successfully built and trained a K-Nearest Neighbors (KNN) classification model capable of accurately identifying iris flower species. The project achieved 100% accuracy on the test dataset and provided valuable insights through data visualization and exploratory data analysis.
